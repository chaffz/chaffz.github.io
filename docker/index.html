基本概念

Docker 包括三个基本概念：

  • 镜像（Image）
  • 容器（Container）
  • 仓库（Repository）

理解了这三个概念，就理解了 Docker 的整个生命周期。

安装

Docker 分为 stable test 和 nightly 三个更新频道。 官方网站上有各种环境下的 “安装指南”，这里主要介绍 Docker 在 Linux 、Windows 10 和 macOS 上的安装。

  • CentOS
  • Debian
  • Ubuntu
  • Mac
  • Windows 10

PS: 因为强哥的问题，所以还需要关注《镜像加速器》。

镜像

Docker 运行容器前需要本地存在对应的镜像，如果本地不存在该镜像，Docker 会从镜像仓库下载该镜像。以下内容需要重点学习：

  • 获取镜像
  • 管理镜像： 列出本地镜像 | 删除本地镜像 | Post not found: docker/image/commit 提交本地镜像 | 创建本地镜像 |
  • 定制镜像： COPY | ADD | ENV | ARG | LABEL | USER | WORKDIR | VOLUME | EXPOSE | CMD | ENTRYPOINT | HEALTHCHECK
